# DHT11-Temparature-and-Humidity-sensor-Arduino 


## DISCRIPTION 
Using DHT11 temperature and humidity sensor to measure the temperature and humidity with Arduino. 

## Components required  
*DHT11 sensor module 
*Arduino UNO 
*Jumper wires 
*Bread board 

## Connections 
Arduino UNO |  DTH11 
-----------|------------- 
5V     |    Vcc 
GND    |    GND 
D4     |    Signal
